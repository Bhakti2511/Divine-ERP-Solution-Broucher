Key Features Mobile App for Customer Inventory Production Management Accounts Other Features
Key Features
  • Mobile App for customers and suppliers.
  • Order Management of customers and suppliers.
  • Barcode & QR Code for product.
  • Photo, Location and Cost-calculation can be stored with product.
  • Accounting
  • Inventory Management
  • Production and Cost Management
  • Accounts for Job-Worker
  • Stock management for Raw-material, Semi-finish and Finish goods.
  • Multi user management with user rights.
  • Divine ERP is totally Cloud based. So, user can access anywhere, anytime, from any device (Laptop, Mobile, Tablet etc.).
  • Life time free updates.
Mobile App for Customer
  • Everything shown in ERP like Products, Price, Stock, Photos etc. are also shown in the Mobile App. No need to manage anything separately in the Mobile App. All settings required for the Mobile App can be done from ERP.

Customer Management

  • Registration is compulsory for new Customer. New customer has to provide Name, Firm name, State & City. Customer can also upload his visiting card.
  • New customer can access mobile app only after approval.
  • Different options can be set while approving, like..
    • Does customer has rights to use application for the life time or for the limited time?
    • Can customer see sale-rate or not?
    • Can customer share product with others or not?
    • Can customer take screenshots or not?
    • Can customer see stock or not?
    • Can customer see only ledger or not?
  • More than one person can place an order from the same firm.
  • User can approve or block the customer.

Settings

  • User can decide whether out-of-stock product should be displayed or not.
  • Banners can be set in app, like fastival ad, any scheme, starting of new product range etc.
  • User can set different information in About-us like Company Name, Address, Contact Details, GST No., Bank details etc.

Order Booking

  • Customers will not be able to book orders more than stock, as well as user can not sell product which is booked.
  • Order can be booked for incomplete box also.
  • Booking from the Mobile App is based on stock management done in ERP. Hence, no need to manage stock beteen Mobile App and ERP differentely.
  • Selected products can be seen in the cart with Qty, Price and Amount. Order can be placed directly from the Cart.

Categories

  • Product Categories can be shown based on name or decided sequence.
  • Product Categories can be shown with Photo.
  • Categories and Sub-categories can be managed.

Products

  • Customer can see new arrivals.
  • Customer can use different filters while seeing the product list. Like..
    • Platting: Gold, Silver, Oxidized
    • Gender: Ladies, Gents, Kids
    • Color: White, Black, Maroon, Green
    • Line: 1-Line, 2-Line, 3-Line
  • User can manage different filters for different categories like Rings, Bichiya, Bangles, Pendant etc.
  • Customer can sort product list based on Latest Products, Price - Low to High and Price - High to Low.
  • Information like Image, Unit, Units/Packing, Price etc. can be seen with the product.

Other Features

  • Customer can see past orders with details.
  • Products from the past orders can be directly added to the cart.
  • Customers can see their Ledger.
  • New product notification can be sent to all customers.
Mobile App for Supplier
  • Raw-material Suppliers and Job-Workers can see pending orders.
  • Pending orders can be seen with Photo & Qty.
  • Suppliers can see their Ledger.
Inventory Management
  • Stock of Finish, Semi-finish and Raw materials can be managed.

Product Categories

  • Units and taxes can be set while adding the product. The suggested units and taxes that have been set in the category, will be set in the product.
  • Prefix and Suffix can be used to generate unique Design No (Model No.).
  • Photo can be set for each Category.
  • In the Mobile App, Categories will be seen with photo. So, no need to manage categories in the Mobile App.
  • User can decide which categories should be displayed in the Mobile App.
  • Categories and Sub-categories can be managed.

Products

  • Product can be managed according to the category.
  • New Design No. and Model No. will be suggested after selecting the category.
  • Packing-Unit and Stock-Unit can be set for the product. E.g. Box & PCs, Box & Pair, Box & Dozen etc.
  • Stock can be managed for Packing-Unit also.
  • Barcode number will be generated automatically.
  • While adding existing product, its barcode can also be added.
  • Opening Stock and Minimum Stock can be added.
  • User can decide whether to take the order for the limited Qty or unlimited Qty.
  • USer can decide whether to sell the product online or not.
  • Multiple photos can be set for product.

Sales Cycle

  • Customers can place online order through Mobile App.
  • Sales Order can be entered manually in the ERP.
  • Pending Sales-Orders can be seen in Product Outstanding report.
  • Sales Invoice can be adjusted against Sales-Order. So, dispatchable product outstanding can be managed.
  • User can not sell stock which is booked.
  • Sales Invoice can be prepared by using Barcode or QR code. While scanning Design No. in barcode, Qty can also be scaned.
  • While using the ERP through mobile, User can scan barcode through barcode-scanner as well as camera.
  • Print option will be displayed after saving.
    • User can print Goods-chit or download the pdf. It shows Party-name, Design No., Product photo, Qty, Location and Total Qty. It is highly helpful for staff to find the goods from rack/location.
    • User can print Invoice on the A4/A5 sized paper.

Purchase Cycle

  • Purchase Order can be generated by using Order Generator tool.
  • Purchase Order can be entered manually in the ERP.
  • Pending Purchase Orders can be seen in Product Outstanding report.
  • Purchase Invoice can be adjusted against Purchase Order. So, receivable product outstanding can be managed.
  • Barcode / QR Code stickers can be printed after saving the purchase entry.

Inhouse Production

  • User can add stock of finish goods with the help of inhouse production entry.
  • After saving, Barcode / QR Code stickers can be printed.

Order Generator

  • Calculate new-order Qty on the basis of Dispatchable, Stock, Minimum Stock & Receivable.
  • Purchase order can be generated for the suppliers.
  • Work order can be generated for Job-worker or Inhouse production.
  • It also calculates need of raw-material for production. Purchase order can be generated for the Raw-material Suppliers.

Bill/Party/Product wise Profit

  • User can see Bill wise profit in the report.
  • User can see Party wise profit in the report.
  • User can see Product wise profit in the report.

Reporting & Other

  • Bill wise Outstanding
  • Stock Register can be seen with Category, Product, Stock and Cost.
  • GS Registers shows details of Sales, Purchase, Sales Order, Purchase Order, Production etc.
Production Management
  • Entry for production of any type of imitation jewellery can be entered like; Bangles, Rings, Bichiya, Mala, Diamond Jewellery, Bali, Pendant etc.

Processes

  • User can define different Processes. Like...
    • Casting
    • Nachak
    • Soldering
    • Buffing
    • Chholkaam
    • Platting
    • Diamond fitting
    • Mina-work
    • Packing
  • Any process can be done within the company or provide outside to Job-worker.

Production

  • Following things will be managed during the entry:
    • Production: OK Qty, Rejection, Product weight, Total weight, Cost etc. can be managed.
    • Consumption: Stock, Consumption Qty, Product weight, Total weight, Cost etc. can be managed.
    • Service: Service, Qty, Rate, Amount etc. can be managed.
    • By Product: Qty, Product weight, Total weight, Cost etc. can be managed.
    • Summary of Weight, Cost and Total Amount.

Job-Work Issue

  • Following things will be managed during the Issue of Job-Work:
    • Receivable Product: Qty, Product weight, Total weight etc. can be managed.
    • Material: Stock, Qty, Product weight, Total weight, Cost etc. can be managed.
    • Weight Summary.
  • Receivable Product Outstanding will be managed for the Job-Work.

Job-Work Receive

  • Following things will be managed during the receive Job-Work:
    • Production: Receivable Qty, OK Qty, Rejection, Difference, Product weight, Total weight, Cost etc. can be managed.
    • Consumption: Party-Stock, Consumed Qty, Return Qty, Rejection Qty, Lost Qty, Product weight, Total weight, Cost etc. can be managed.
    • Services: Servie, Qty, Rate, Amount etc. can be managed.
    • By-Products: Qty, Product weight, Total weight, Cost etc. can be managed.
    • Summary of Weight, Cost and Amount.
    • Calculation of payable amount to Job-worker.
  • Job-Work Received will be adjusted against Job-Work Issued.
Account

Receipt

  • User can prepare Cash Receipt & Bank Receipt separately.
  • કેશ ડીસકાઉન્ટ પણ આપી શકાય છે.
  • Cash discount can also be provided.
  • Previous Balance, Amount & Net Balance can be included in Receipt-print.

Payment

  • User can prepare Cash Receipt & Bank Receipt separately.
  • Cash discount received can also be entered.

Outstanding (Receivable/Payable)

  • Party and Closing Balance can be shown.
  • Advance received / payable from the Party is shown separately.

Ledger

  • User can see ledger of multiple patries in a single report.
  • User can see ledger between any date range (No limit of financial year).

Trading Report

  • USer can see Trading Report between any date range (No limit of financial year).

Profit & Loss Report

  • User can see Profit & Loss Report between any date range (No limit of financial year).

Balance Sheet

  • User can see Balance Sheet of any date (No limit of financial year).
Other Features

Databases

  • User can create multiple databases.

Multi User management

  • Multi user can work on the same database.
  • Different rights can be provided to different users. Like..
    • User can Add, Edit, Delete & Print Invoice.
    • On-Screen View, Print-Preview, Download-PDF & Download-Excel for the Report.
    • Add, Edit & Delete rights can be provided to the users for different setups like Products, Contacts, Accounts etc.
  • Different database rights can be given to different users.

Backup

  • User can take backup and download to the local computer.
